# Java Data Structures Suite
Four data-structures and algorithms projects in Java, each pairing a working implementation with formal Big-O complexity analysis.

# Overview
A collection demonstrating core CS fundamentals: custom data structures, object-oriented design, and algorithmic reasoning. Each project includes complexity proofs alongside the implementation.

# Projects
Polynomial — a singly-linked-list polynomial class supporting addition, multiplication, and evaluation.
ChangeStack — a custom stack supporting bottom-k increment/decrement operations.
Triage Simulator — a hospital triage simulation using three priority queues to order patients by severity.
Pizza Builder — an object-oriented pizza-ordering model demonstrating composition and encapsulation.

Each is accompanied by formal Big-O complexity proofs.

# Tech stack
Java · Data Structures · Algorithms · OOP


# Run it
 from each project folder:

javac *.java

java Main
